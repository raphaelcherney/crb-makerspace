# CNC Milling Machine

The CRB Makerspace Shop has a Carvera CNC milling machine. You must complete machine-specific training before being authorized to use this machine. **Do not cut steel on this machine.**

## Hazard level

<span style="background-color: orange; color: white; padding: 4px 8px; border-radius: 4px; font-size: 2.2em; font-weight: bold;">MEDIUM</span>

!!! warning

    - High-speed cutting tool contact can cause injury if enclosure is opened during operation
    - Flying chips and metal fragments when opening enclosure
    - Risk of unexpected machine movement from programming errors
    - Pinch points from moving axes
    - Hot cutting tools and workpiece after operation

## Safety

- **DO NOT open enclosure while machine is running**
- **NEVER reach into machine during operation**
